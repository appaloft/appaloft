# Appaloft Brand Assets

These SVG files are the canonical Appaloft product brand assets for Web, Docs, and future www
surfaces.

## Included Files

- `appaloft-icon-light.svg`: light rounded-square app icon.
- `appaloft-icon-dark.svg`: dark rounded-square app icon.
- `appaloft-logo-horizontal.svg`: mark with dark wordmark.
- `appaloft-logo-horizontal-inverse.svg`: mark with white wordmark.

Use these through package exports such as
`@appaloft/design/assets/appaloft-icon-light.svg`.
