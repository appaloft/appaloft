<h2 id="self-hosting-serve-paths">Serve 路径</h2>

`appaloft serve` 根路径提供 Web console，`/docs/*` 提供 public docs。

<h2 id="self-hosting-install-binary">安装 binary</h2>

Binary 包含运行时入口、Web console 静态资源和 public docs 静态资源。
