## Generated operation pages [#openapi-generated-reference]

The public docs site generates a dedicated reference page for every HTTP operation from the Appaloft OpenAPI 3.1 schema.

Those generated operation pages live under `/docs/reference/openapi/*`, while `/docs/en/reference/openapi/` remains the stable English entrypoint for Web help, CLI help, and external bookmarks.

## Contract source of truth [#openapi-source-of-truth]

The OpenAPI schema is generated by `@appaloft/openapi`. Operation names, input schemas, output shapes, and error descriptions must continue to come from the shared business operation contracts.

When API behavior changes, update the business operation catalog, public docs coverage records, and corresponding API/CLI help links in the same change.
