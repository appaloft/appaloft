<h2 id="reference-error-shape">错误形状</h2>

用户可见错误应包含稳定 code、category、phase 和可执行恢复建议。

<h2 id="reference-status-shape">状态形状</h2>

状态应区分资源、部署、运行时、代理、访问地址和证书 readiness。
