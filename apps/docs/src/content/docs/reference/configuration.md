<h2 id="reference-runtime-configuration">运行时配置</h2>

运行时配置控制 Appaloft serve、数据库、静态资源目录和自托管行为。

<h2 id="reference-docs-static-dir">Docs 静态资源目录</h2>

`APPALOFT_DOCS_STATIC_DIR` 覆盖 public docs 静态资源，不覆盖 Web console。
